# 딥러닝을 활용한 도로 포트홀 분류 및 탐지모델

## 프로젝트 개요
- 최근 여름철 폭우 또는 겨울철 폭설로 인한 도로 노면 이상 다수 발생
- 특히 도로 위 구멍, 포트홀에 의한 타이어 손상 및 급작스러운 조향에 따른 사고 위험
- **프로젝트 목표 : 이미지나 영상의 도로에서 포트홀을 감지하는 분류 모델, 탐지 모델 생성**
